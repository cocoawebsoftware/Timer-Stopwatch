# Modern Timer & Stopwatch / モダンタイマー・ストップウォッチ

A fully HTML-based **Timer / Stopwatch** with modern UI design.  
HTMLだけで完結した、モダンUIの **タイマー / ストップウォッチ** です。

---

## Features / 機能

- Timer and Stopwatch modes / タイマーとストップウォッチモード  
- Start/Stop unified button / スタート・ストップを一つのボタンで操作  
- Lap functionality for Stopwatch / ストップウォッチでラップ記録可能  
- Decimal display for Stopwatch / 小数表示対応  
- Timer adjustable with up/down arrow buttons (minutes & seconds) / タイマーは上下矢印で分・秒を調整可能  
- Long press acceleration on arrow buttons / 長押し加速対応  
- Dark / Light mode toggle / ダーク・ライトモード切替  
- Modern UI: rounded corners, shadows, hover effects / モダンUI（角丸・影・ホバーエフェクト）  
- Numbers use **Gill Sans Light** (`font/Gill Sans Light.otf`) / 数字は **Gill Sans Light** フォント使用  
- Fully self-contained in **one HTML file** / HTML1ファイルで完結  
- Sound alarm on timer finish / タイマー終了時に音アラーム  

---

## Usage / 使い方

1. Clone or download this repository / リポジトリをクローンまたはダウンロード  
2. Place the `Asset` folder with SVGs and `font/Gill Sans Light.otf` in the same directory as the HTML / `Asset`フォルダと `font/Gill Sans Light.otf` を HTML と同じディレクトリに配置  
3. Open `index.html` in your browser / ブラウザで `index.html` を開く  
4. Use the **switch mode** button to toggle between Stopwatch and Timer / 「モード切替」ボタンでストップウォッチとタイマーを切替  
5. Adjust the timer using arrow buttons (only when the timer is stopped) / タイマーは停止中に矢印ボタンで分・秒を調整  
6. Use the **Dark Mode** toggle to switch themes / 「ダークモード切替」ボタンでテーマ切替  
7. Click **Start** to run Timer/Stopwatch. Stopwatch allows **Lap** tracking / 「スタート」ボタンでタイマー/ストップウォッチ開始。ストップウォッチではラップ記録可能  

---

## Browser Compatibility / ブラウザ互換性

- Chrome, Firefox, Edge: fully supported / 完全対応  
- Safari: `.otf` font may have limited support / Safariでは.otfフォントのサポートが限定的
